# Destination Images

This folder contains destination background images for the Swanubhav website.

## Required Images

Please add the following images to this folder:

- `bali.jpg` - Bali beach sunset (1920x1080 or larger recommended)
- `kashmir.jpg` - Kashmir mountain / Dal Lake (1920x1080 or larger recommended)
- `paris.jpg` - Paris cityscape (1920x1080 or larger recommended)
- `tokyo.jpg` - Tokyo skyline (1920x1080 or larger recommended)
- `santorini.jpg` - Santorini sunset (1920x1080 or larger recommended)
- `maldives.jpg` - Maldives overwater bungalows (1920x1080 or larger recommended)

## Image Specifications

- **Format**: JPG or WebP
- **Recommended Size**: 1920x1080px or larger
- **Aspect Ratio**: 16:9 or similar landscape orientation
- **Quality**: High resolution for crisp display on all devices
- **File Size**: Optimized for web (under 500KB recommended)

## Usage

These images are used as background images throughout the site:
- Home page hero section
- Background sections between content
- Featured destination cards

The images will have a gradient overlay applied automatically to ensure text readability and brand consistency.

## Adding More Destinations

To add more destinations:
1. Add the image file to this folder
2. Update `/lib/destinations.ts` with the new destination data
3. The component will automatically use it!
